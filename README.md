# terraform-handson
ローカルでterraformを試そう

## コマンド一覧

|  | 説明 |
|-------------|-----------------------------|
| 実行環境の作成 | `docker compose up -d` or `docker-compose up -d` |
| 初期化 | `terraform init` |
| 実行計画の確認 | `terraform plan` |
| リソースの作成・変更 | `terraform apply` |
| ロック解除 | `terraform force-unlock` |
| 実行環境の削除 | `docker compose down` or `docker-compose down` |
